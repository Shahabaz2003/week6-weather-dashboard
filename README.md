# 🌤️ Weather Dashboard Application

A comprehensive CLI-based weather application that fetches real-time weather data and 5-day forecasts from the OpenWeatherMap API.

## 🚀 Features
- 📍 **Current Weather:** View temperature, humidity, wind speed, pressure, and visibility.
- 📅 **5-Day Forecast:** Detailed daily summary with min/max temperatures and conditions.
- 🔄 **Unit Conversion:** Easily switch between Celsius and Fahrenheit.
- ⚡ **API Response Caching:** Caches request data locally for 10 minutes to minimize API consumption.

## 📦 Tech Stack & Libraries
- **Language:** Python 3.9+
- **Libraries:** `requests` (for API calls), `python-dotenv` (for secure environment configuration)
